Gemini Live Recorder V3 Audio Only

Speak into the mic, let Gemini respond, then stop the encounter and download one WAV file with the full conversation audio. This mode disables live transcript handling and keeps the session audio-only to reduce local latency.
Idle
This is sent in the session config. Change it before starting a new encounter.
Notes
1. This page sends your key directly from the browser. Fine for testing, not ideal for production.
2. For production, use ephemeral tokens from your backend.
3. The downloaded file is a mixed WAV containing both your microphone and Gemini's audio replies.
4. Live transcript is disabled in this version to reduce overhead.
5. Gemini audio is always included in the WAV even if speaker playback is turned off.
6. This file imports @google/genai from a CDN at runtime, so internet access is required.
